# FrontDesk-Desktop
FrontDesk - Cross-platform desktop app for event registrations, ticketing and transactions

## Requirements
 - [Node.JS](https://nodejs.org/en/)

## Installing
 - `cd /path/FrontDesk-Desktop`
 - `npm i`

 ## Running
 - `npm start`
